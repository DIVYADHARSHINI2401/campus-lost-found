# Campus Lost & Found Website

A Django web application for college students to report and find lost and found items.

## Features

- User registration and authentication
- Post lost or found items
- Upload item images
- Search items
- Contact owners

## Technologies

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite

## Setup

1. Install Python and create virtual environment.
2. Install dependencies: `pip install django pillow`
3. Run migrations: `python manage.py makemigrations` then `python manage.py migrate`
4. Run server: `python manage.py runserver`

## Usage

- Signup/Login
- Post items
- Search and view details
- Contact via email/phone
